Getting Started

Tabby is a set of lightweight toggle tabs powered by vanilla JS.

  1. Basic. Simple, unstyled toggle tabs.
  2. Horizontal. Styled navigation that displays horizontally.
  3. Stacked. Vertical navigation next to the content.

Browser Support

Tabby works in all modern browsers, and IE 9 and above.

If you need to support older browsers, you can still download the jQuery version of Tabby on GitHub.

Built by Chris Ferdinandi. Free to use under the MIT License.

Tabby is an add-on for Kraken, a lightweight, mobile-first boilerplate for front-end web developers. It works great as a standalone product, too.
