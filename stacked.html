How to use Tabby Stacked

1. Include Tabby on your site.

<link rel="stylesheet" href="tabby.css">
<script src="tabby.js"></script>

If you don't already have a grid system, also include kraken-grid.css.

2. Add the markup to your HTML.

<div class="row">
    <div class="grid-2">
        <div class="tabs">
            <li class="active"><a data-target="#tab1" href="#">Superheroes</a></li>
            <li><a data-target="#tab2" href="#">Ice Cream</a></li>
            <li><a data-target="#tab3" href="#">Seasons</a></li>
        </div>
    </div>
    <div class="grid-4">
        <div class="tabs-content">
            <div class="tabs-pane active" id="tab1">
                Superheros
                ...
            </div>

            <div class="tabs-pane" id="tab2">
                Ice Cream
                ...
            </div>

            <div class="tabs-pane" id="tab3">
                Seasons
                ...
            </div>
        </div>
    </div>
</div>

Tab toggles can be buttons or links. Make sure that the data-target value of each tab toggle matches the ID of the matching .tab-pane. Add the .active class to the tab and content that you'd like displayed by default.

Tabby Stacked uses the grid from Kraken to provide structure. Feel free to replace with your own.

3. Add a feature test.

Tabby uses modern JavaScript API's that aren't supported by older browsers. Add this short snippet of code in the <head> of your HTML document to feature test:

<!-- Feature Test -->
<script>
    if ( 'querySelector' in document && 'addEventListener' in window && Array.prototype.forEach ) {
        document.documentElement.className += 'js';
    }
</script>

For browsers that support modern JavaScript API's, this snippet adds a .js class to the <html> element, which activates Tabby. Older and less capable browsers will be shown all tab content by default, and the navigation tabs will be hidden.

4. Pat yourself on the back.

Seriously, you're done. Nice work! Modify as needed.
